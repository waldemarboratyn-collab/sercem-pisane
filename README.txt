Instrukcja:
- Wypakuj zawartość i wrzuć do repozytorium root (branch main).
- Struktura:
  - index.html
  - style.css
  - wiersze/
      - index.html (lista)
      - 1.html ... 117.html (podstrony)
  - wiersze.json (metadane)
- Jeśli masz treści pełne, zastąp pole 'content' w wiersze.json lub edytuj odpowiednie pliki w wiersze/*.html.
