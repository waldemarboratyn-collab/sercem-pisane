
Sercem pisane bazgrołki— Multi-page pastelowy projekt (lazur + piasek)

Instrukcja krótkia:
1. Pobierz i rozpakuj tę paczkę.
2. Wejdź do repo: https://github.com/waldemarboratyn-collab/sercem-pisane
3. Kliknij Add file → Upload files i wgraj wszystkie pliki (upewnij się, że trafiają do root repo).
4. Commit changes.
5. Settings → Pages → Source: main, Folder: / (root) → Save.
6. Poczekaj ~30-60s, odśwież stronę: https://waldemarboratyn-collab.github.io/sercem-pisane/
